Color Schemes for lepton-schematic


Schematic for testing color schemes: cstst.
The color boxes with hex color values are created with cstst.scm script (download), using cstst-mk() function. Call cstst-up() to update hex color values to match currently loaded color scheme.

To use a color scheme in lepton-schematic, add ( primitive-load "/path/to/color_scheme_file" ) to your gschemrc configuration file.

NOTE: Currently, lepton-eda and geda-gaf use the same file format for color schemes.
Feel free to send me your custom lepton/geda color schemes, I will publish them here.
