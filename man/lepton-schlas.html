NAME

lepton-schlas − Lepton EDA schematic load and save

SYNOPSIS

lepton-schlas [−e|−u] [−q|−v] file [ file ... ]

lepton-schlas -h

DESCRIPTION

lepton-schlas can embed the in-use schematic symbols and pictures into a schematic or set of schematics. It allows, for instance, putting a schematic on a website or in an email without the recipient needing to download lots of custom symbols for the design.

lepton-schlas can also unembed symbol references from a schematic, but note that the symbols revert to library references; if the symbol doesn’t exist in the current Lepton EDA project’s library path, a broken schematic results.

OPTIONS

−e, −−embed

Embed all components/pictures.

−u, −−unembed

Unembed all components/pictures.

−q, −−quiet

Quiet mode.

−v, −−verbose

Verbose mode.

−h, −−help

Print usage information.
