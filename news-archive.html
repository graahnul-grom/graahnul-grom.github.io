News archive

November, 24, 2020:
Lepton EDA reference manual has been added to the manuals section.
It will be included in upcoming Lepton EDA release 1.9.12.

June, 23, 2020:
A new Guile script for lepton-schematic has been added: select-locked.scm.
Select locked component under the mouse cursor, cycle through all locked symbols, or select all of them. After that it's possible to either unlock the selected object or edit its properties without unlocking.
