14.2.9 pinlabel attribute

This attribute labels a pin object. This attribute is primarily used by lepton-netlist to support hierarchical designs.

This attribute must be attached to the pin and be left visible. Please make this attribute green (instead of the default attribute yellow).

Examples: 

pinlabel=A0
pinlabel=DATA1
pinlabel=CLK
