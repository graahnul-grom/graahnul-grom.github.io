9 lepton-sch2pcb - interface to PCB layout programs

lepton-sch2pcb is a bridge between lepton-schematic and PCB layout programs such as pcb or pcb-rnd. It is Lepton’s successor to gsch2pcb in the geda-gaf project. The program is used to forward annotate changes in schematics to PCB layout programs so that the schematics are always in sync with the layout created from them. Additions and deletions of elements in the design layout as well as changes in their interconnections are automatically driven by changes in the design schematics.
