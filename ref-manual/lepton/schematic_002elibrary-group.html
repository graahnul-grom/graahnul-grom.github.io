3.4.1.8 schematic.library group

Since 1.9.10. Before 1.9.10 it was gschem.library group.

NameTypeDefault valueDescription
component-attributes‘string list’‘*’List of attribute names (semicolon-separated) that are displayed in the component select dialog. An empty list will disable the attribute view. If the first list element is an asterisk (‘*’), all attributes will be displayed in the alphabetical order.
sort‘boolean’‘false’If ‘true’, the component libraries are sorted alphabetically. Otherwise they are sorted in the order opposite to what they were added in.
