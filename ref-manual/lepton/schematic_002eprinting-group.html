3.4.1.9 schematic.printing group

Since 1.9.10. Before 1.9.10 it was gschem.printing group.

NameTypeDefault valueDescription
layout‘string’‘auto’Possible values: ‘auto’, ‘portrait’, ‘landscape’. When using a paper size, set the orientation of the output. If ‘auto’ layout is used, the orientation that best fits the drawing will be used.
‘monochrome’‘boolean’‘false’Toggle monochrome (‘true’) or color (‘false’) output.
paper‘string’‘"iso_a4"’Size the output for a particular paper size. The default value depends on the current locale. Described in the PWG 5101.1 standard (Printer Working Group "Media Standardized Names"). Examples: ‘iso_a4’, ‘iso_a5’, ‘na_letter’.
