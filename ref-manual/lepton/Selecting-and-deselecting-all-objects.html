5.11.3 Selecting and deselecting all objects

Edit → Select All (Ctrl-A) selects all non-locked objects on a page while Edit → Deselect (Shift-Ctrl-A) deselects all selected objects, if any. The latter can be achieved by just left mouse click at some free space of schematic.
