12.3.2 graphical attribute

Symbols which have no electrical or circuit significance need a graphical=1 attribute. Symbols like titleboxes are purely graphical symbols. Any symbol which has graphical=1 is ignored by lepton-netlist.

graphical=1 should exist somewhere in the symbol and made invisible. This is a free standing or floating attribute. Don’t forget to set device=none (device attribute).

Example: 

graphical=1
